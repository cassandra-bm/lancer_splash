Links

Here I've compiled some useful resources for anyone looking to play or host Lancer. If you have any burning recommendations please reach out!

  • Lancer Core Book - Everything you need to know - free version here!
  • Lancer Cards - Quick reference for almost all lancer rules!
  • COMP/CON - Online character sheet editor and knowledge compendium. It's modular!
  • RetroGrade Minis - Create beautiful custom sprite art for PC and NPC tokens! Very free, but requires a wee Patreon subscription for export convenience.
  • Lancer Map Creation Tool by Interpoint - A popular map editor I've seen floating around. Most first party modules come with pre-made mapsets.
  • Foundry Virtual Tabletop - Recommended alternative to Roll20. Not free. Requires elbow grease.
